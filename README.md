#Shoestring
An attempt to use SASS to recreate mini grid system inspired by Twitter Bootstrap

###Technologies
#####Node
* Express
* Swig
* Node-Sass

####HTML 5

####CSS 3
* Normalize
* SASS

##How to use
1. Clone this repo
2. `cd` into root directory and do `npm install`
3. To run, use `npm start`
4. Go to `localhost:3000` on your browser
